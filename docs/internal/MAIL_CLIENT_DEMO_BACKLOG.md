# Mail Client Demo Backlog

This document outlines the backlog for a future, independent showcase example: `examples/mail-client-demo`.

The Mail functionality was previously part of the `communication-suite-demo` but was removed to focus that demo purely on a cohesive Adaptive Chat product experience.

## Future Demo Location
`examples/mail-client-demo`
Route: `/examples/mail-client/`

## Planned Features
The future Mail example should eventually contain the following features to demonstrate AdaptiveKt's capabilities in a complex, data-heavy layout:

- **Folders / Navigation**: Inbox, Starred, Sent, Drafts, Archive, Spam, Trash, Labels.
- **Search**: Advanced filtering and search mockups.
- **Reading Pane**: Responsive two-pane and three-pane layouts for viewing threads.
- **Conversation Threads**: Clean rendering of long email threads and historical replies.
- **Compose**: New message modal and inline reply composer with attachments.
- **Contacts Suggestions**: Mock auto-complete for "To", "Cc", "Bcc" fields.
- **Scheduled Sending**: Mock scheduling interactions.
- **Signatures**: Settings for managing user signatures.
- **Responsive Flows**: 
  - Mobile mail flow (List -> Detail stack).
  - Tablet/Desktop split panes (List + Detail side-by-side).
- **Theming**: Real light and dark visual validation.
