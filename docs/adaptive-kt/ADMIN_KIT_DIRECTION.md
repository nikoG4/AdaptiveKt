# AdaptiveKt: Dirección de Producto

AdaptiveKt evoluciona de una librería de patrones adaptativos a un Admin Kit profesional para Compose Multiplatform, inspirado en AdminLTE, CoreUI y AdminKit, pero idiomático para Kotlin/Compose.

## Principios
- Default simple code should produce a professional admin UI.
- Professional defaults: sin necesidad de 20 modifiers para un resultado visual admin.
- Responsive patterns: no solo tamaños, sino patrones de navegación, layout y feedback adaptativos.
- Foundation-only: sin dependencia obligatoria de Material 3.
- No dependencia obligatoria de icon pack.
- Admin-demo como integración visual y validación de capturas.

## Objetivo
Ofrecer componentes y patrones admin con buenos defaults, para que el developer obtenga una UI profesional con el mínimo código y configuración.
