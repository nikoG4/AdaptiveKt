# Support

AdaptiveKt is maintained as an open-source alpha project. Support is best-effort and focused on improving the library for the community.

## Where To Get Help

- Documentation: https://nikog4.github.io/AdaptiveKt/
- GitHub Issues: use issues for bugs, regressions, documentation problems, and concrete feature requests.
- GitHub Discussions: use discussions if enabled for broader questions and design exploration.

## Good Support Requests Include

- AdaptiveKt version.
- Module or component involved.
- Target platform.
- Kotlin, Compose Multiplatform, and Gradle versions.
- A small reproduction or code snippet.
- Screenshots for visual issues.

## Limits

The project cannot provide private integration support, guaranteed response times, or debugging for unrelated app architecture. Commercial production support is not currently offered.
