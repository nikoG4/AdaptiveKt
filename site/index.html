AdaptiveKt

Compose Multiplatform Admin UI Toolkit

Build responsive dashboards, data views, forms, navigation shells, and shared components across Desktop, Android, and Web/Wasm.

View Demo Read Docs

Features

Adaptive breakpoints

Compact, medium, expanded, and large behavior for admin screens.

Layouts and grid

Container and grid primitives for responsive composition.

Navigation scaffold

Breakpoint-aware admin navigation without owning app routing.

Data and forms

Responsive table/card data views and structured form layout.

Components

Buttons, badges, avatars, thumbnails, chips, cards, fields, menus, and select.

Visual verification

Desktop capture with Robot and Web capture with Playwright.
