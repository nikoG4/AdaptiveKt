Features

🎯 Adaptive Layout System

Breakpoint-aware components that respond to screen size. Automatically adapts layouts for Compact (phone), Medium (tablet), Expanded (tablet landscape), and Large (desktop) screens.

📊 Enterprise Data Views

Responsive table/card hybrid that switches layouts based on available space. Built-in filtering, sorting, row selection, and custom actions.

📋 Professional Forms

Adaptive form layouts with intelligent label positioning. Validation states, multi-column support, and responsive field organization.

🧭 Smart Navigation

Navigation scaffold that adapts surfaces by breakpoint. Drawer on phones, Bottom Navigation on tablets, Rails on landscape, and Sidebar on desktop.

🎨 Rich Component Library

Button variants, badges, avatars, cards, menus, text fields, search inputs, and more. All with consistent styling and adaptive sizing.

🌐 Multiplatform Ready

Works on JVM/Desktop (Compose), Android, iOS, and Web/Wasm. One codebase for all platforms.

Quick Start

Add to your Gradle project:

// settings.gradle.kts
include(":adaptive-kt:adaptive-core")
include(":adaptive-kt:adaptive-components")

// build.gradle.kts
dependencies {
    implementation(project(":adaptive-core"))
    implementation(project(":adaptive-components"))
    implementation("org.jetbrains.compose.foundation:foundation:1.8.2")
}

Use in your UI:

@Composable
fun MyApp() {
    val adaptiveInfo = rememberAdaptiveInfo()
    
    AdaptiveContent(adaptiveInfo) {
        AdaptiveButton("Save", onClick = { })
        AdaptiveDataView(items, columns)
    }
}
