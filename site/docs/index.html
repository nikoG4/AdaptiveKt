Installation

AdaptiveKt is not yet published to Maven Central. For now, use it as a source dependency.

// settings.gradle.kts
include(":adaptive-kt:adaptive-core")
include(":adaptive-kt:adaptive-components")
include(":adaptive-kt:adaptive-layout")
include(":adaptive-kt:adaptive-navigation")
include(":adaptive-kt:adaptive-forms")
include(":adaptive-kt:adaptive-data")
include(":adaptive-kt:adaptive-feedback")

Quick Start

Create a responsive admin layout in minutes:

@Composable
fun AdminApp() {
    val adaptiveInfo = rememberAdaptiveInfo()
    
    AdaptiveNavigationScaffold(
        adaptiveInfo = adaptiveInfo,
        navigationItems = listOf(
            AdaptiveNavItem("home", "Dashboard", Icons.Filled.Dashboard),
            AdaptiveNavItem("users", "Users", Icons.Filled.People),
        ),
        selectedId = "home",
        onNavigate = { /* handle nav */ },
    ) { navHeight ->
        Box(modifier = Modifier.padding(bottom = navHeight)) {
            AdaptiveContent(adaptiveInfo) {
                Text("Your content here")
            }
        }
    }
}

Breakpoints

AdaptiveKt uses four standard breakpoints to adapt layouts:

  • Compact — < 600 dp (phone portrait)
  • Medium — 600–840 dp (tablet portrait)
  • Expanded — 840–1200 dp (tablet landscape)
  • Large — ≥ 1200 dp (desktop)

Adaptive Info

Get current breakpoint and window information:

val adaptiveInfo = rememberAdaptiveInfo()
println(adaptiveInfo.breakpoint)  // Current breakpoint
println(adaptiveInfo.windowWidth) // Window width in dp

Responsive Layouts

Use breakpoints to build responsive UIs:

@Composable
fun ResponsiveGrid() {
    val adaptiveInfo = rememberAdaptiveInfo()
    
    val columnCount = when (adaptiveInfo.breakpoint) {
        Compact -> 1
        Medium -> 2
        Expanded -> 3
        Large -> 4
    }
    
    AdaptiveGrid(columnCount = columnCount) {
        // Grid items
    }
}

Navigation

AdaptiveNavigationScaffold automatically selects the best navigation UI for each breakpoint:

  • Compact: Drawer
  • Medium: Bottom Navigation
  • Expanded: Navigation Rail
  • Large: Sidebar

Buttons

Multiple button variants for different contexts:

AdaptiveButton("Save", onClick = {}, variant = Primary)
AdaptiveButton("Cancel", onClick = {}, variant = Secondary)
AdaptiveButton("Delete", onClick = {}, variant = Danger)

Forms

Build responsive forms with AdaptiveFormLayout:

AdaptiveFormLayout(columnCount = 2) {
    AdaptiveFormField(label = "Name") {
        AdaptiveTextField(value = name, onValueChange = { name = it })
    }
    AdaptiveFormField(label = "Email") {
        AdaptiveTextField(value = email, onValueChange = { email = it })
    }
}

Data Views

Display tabular data with automatic table/card switching:

AdaptiveDataView(
    items = users,
    columns = listOf(
        DataColumn("Name") { it.name },
        DataColumn("Email") { it.email },
    ),
    state = DataState.Content(users),
)

Theming

Dark mode and custom themes are planned for a future release. Currently, components use adaptive light colors.

Multiplatform

AdaptiveKt works on:

  • JVM/Desktop — Full support via Compose Desktop
  • Android — Full support (test with device rotation)
  • iOS — Target declared (macOS validation pending)
  • Web/Wasm — Full support (try the demo in your browser)
